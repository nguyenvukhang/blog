# Using Hwa Chong's WiFi

Login credentials are your Windows Login ID and password.

Set the following settings and you shouldn't be too far off:  
* **SSID/Network name:** STUDENT_WLAN
* **Security type:** WPA2-Enterprise
* **Encryption type:** AES
* **Network authentication method:** Protected EAP (PEAP)
* **PEAP authentication method:** Secured password (EAP-MSCHAP v2)
* **Validate server certificate:** uncheck
